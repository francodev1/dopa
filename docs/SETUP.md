# Setup

1. Instale Node.js (v18+ recomendado).
2. Copie `.env.example` para `.env` e preencha as chaves.
3. Vá para a pasta `backend` e instale dependências:

```bash
npm install
```

4. Rode o servidor:

```bash
npm start
```

5. Configure seu webhook no painel do Evolution apontando para `/webhook`.
